# TerraVision AI

Enterprise AI-powered land intelligence platform.

TerraVision AI is an AI-powered land intelligence platform that combines property, economic, demographic, infrastructure, and environmental data to predict future land values and provide investment insights. The platform automates data collection, engineering, machine learning, and interactive analytics to help users make informed real estate decisions.

## Tech Stack
Python, SQL, Airflow, Snowflake, dbt, FastAPI, Tableau, Docker, AWS.
